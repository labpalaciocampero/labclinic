# Laboratorio Palacio Campero — Sistema de Análisis Clínicos

Sistema de gestión de análisis clínicos con soporte PWA.

## URL del sistema
https://labpalaciocampero.github.io/labclinic/

## Instalación como app de escritorio
1. Abrí la URL en Chrome
2. Hacé clic en el ícono ⊕ en la barra de direcciones
3. Seleccioná "Instalar Laboratorio Palacio Campero"
4. El ícono aparece en el escritorio

## Actualizar el sistema
El sistema se actualiza automáticamente al recargar la página (F5).
